В Яндекс.
========
Тестовое задание для Школы разработчиков интерфейсов.
========

Сверстана страница для формы-анкеты с заданиями для кандидатов.
Фичи:
* **0x00.**  Страница работает в браузерах  MSIE (8, 9, 10), Google Chrome (26, 27), Firefox (20, 21), Opera (12.15, 12.14), Яндекс.Браузер 1.5

* **0x01.** На странице использованы приемы безопасной деградации

* **0x02.** Страница работает с отключенным JavaScript и/или CSS

* **0x03.** На странице присутствует статус заполнения для первых 15ти вопросов анкеты

* **0x04.** На странице есть окно быстрого заполнения анкеты

* **0x05.** Страница версталась с ипользованием БЭМ

* **0x06.** Страницу можно посмотреть по адресу it-zombie.ru/yandex-test/test.html
